## 安装 Hexo
```
$ npm install -g hexo-cli
```
## Create a new post
```
$ hexo new "My New Post"
```
## Run server
```
$ hexo server
```
## Generate static files
```
$ hexo generate
简写为
$ hexo g
```
## Deploy to remote sites
```
$ hexo deploy
```
## Hexo help 查看所有命令
```
$ hexo help
```
